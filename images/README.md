# Recipe Images Directory

This directory stores uploaded recipe images.

## How to Add Images

### Option 1: Upload via Admin Panel
1. Login as admin
2. Go to recipe management
3. Upload image using the admin interface

### Option 2: Manual Upload
Place image files in this directory with the same filename as stored in the database.
